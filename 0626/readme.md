# 파이썬 컨테이너(Container) 요약

## 컨테이너란?

-   여러 개의 데이터를 하나의 변수에 저장하는 자료구조
-   대표적인 컨테이너: **리스트(List), 튜플(Tuple),
    딕셔너리(Dictionary), 집합(Set)**

  ---------------------------------------------------------------------------------------
  종류                   생성             순서        수정        중복        접근
  ---------------------- ---------------- ----------- ----------- ----------- -----------
  리스트(List)           `[]`             O           O           O           인덱스

  튜플(Tuple)            `()`             O           X           O           인덱스

  딕셔너리(Dictionary)   `{key: value}`   O           O           Key X /     Key
                                                                  Value O     

  집합(Set)              `{}` 또는        X           O           X           불가
                         `set()`                                              
  ---------------------------------------------------------------------------------------

## 언제 사용할까?

-   **리스트**: 순서가 있는 데이터를 저장하고 수정할 때
-   **튜플**: 변경되지 않는 데이터를 저장할 때
-   **딕셔너리**: Key를 이용하여 빠르게 데이터를 찾을 때
-   **집합**: 중복 제거와 집합 연산이 필요할 때

## 예제

``` python
fruits = ["사과", "바나나", "포도"]
point = (10, 20)
student = {"name": "홍길동", "age": 20}
numbers = {1, 2, 3, 2}

print(fruits)
print(point)
print(student["name"])
print(numbers)
```

## 핵심 정리

-   리스트: 가장 많이 사용하는 컨테이너
-   튜플: 수정 불가능한 데이터 저장
-   딕셔너리: Key-Value 형태의 데이터 저장
-   집합: 중복 제거 및 집합 연산
